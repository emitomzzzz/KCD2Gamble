# Dice Game

这是一个双人回合制骰子游戏，保留了原来的计分规则，并新增了 `tkinter` 图形界面。

## 运行方式

启动图形界面：

```bash
python dice_game.py --gui
```

启动命令行版本：

```bash
python dice_game.py
```

也可以通过参数指定目标分数和随机种子：

```bash
python dice_game.py --gui --target 3000 --seed 7
python dice_game.py --target 3000 --seed 7
```

## 图形界面说明

- 顶部可以设置目标分数和随机种子，并随时开始新游戏
- 左侧显示双方总分、当前回合暂存分数和待掷骰子数量
- 中间骰子区支持点击选择得分骰子
- 下方按钮分别用于掷骰、拿走并计分、继续掷剩余骰子、把本回合分数入账
- 右侧包含对局日志和完整计分规则

## 计分规则

- 单个 `1` 记 `100` 分
- 单个 `5` 记 `50` 分
- 三个相同点数：`1` 记 `1000` 分，`2-6` 记点数乘 `100`
- 四、五、六个相同点数：分别是三连分数的 `2 / 3 / 4` 倍
- 顺子 `1 2 3 4 5 6` 记 `1500` 分
- 三对记 `1500` 分
- 两组三连记 `2500` 分
- 四个相同点数加一对记 `1500` 分
