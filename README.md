# Dice Game

这是一个双人命令行摇骰子游戏，按照六颗骰子的回合制玩法实现。

## 运行

```bash
python dice_game.py
```

可以通过参数指定目标分数和随机种子：

```bash
python dice_game.py --target 3000 --seed 7
```

## 默认计分规则

- 单个 `1` 记 `100` 分
- 单个 `5` 记 `50` 分
- 三个相同点数：`1` 为 `1000` 分，`2-6` 为点数乘以 `100`
- 四、五、六个相同点数：分别是三连分数的 `2`、`3`、`4` 倍
- 顺子 `1 2 3 4 5 6` 记 `1500` 分
- 三对记 `1500` 分
- 两组三连记 `2500` 分
- 四个相同点数加一对记 `1500` 分

## 玩法

1. 玩家 A 和 B 轮流进行回合。
2. 每个回合开始时投掷当前剩余的骰子。
3. 如果本次投掷没有任何得分组合，则本回合立即结束，且本回合未存入总分的分数全部作废。
4. 如果有得分组合，玩家选择要拿走并计分的骰子。
5. 玩家可以继续投掷剩余骰子，或者把本回合暂存分数计入总分并结束回合。
6. 当一名玩家达到目标分数时，游戏结束。
