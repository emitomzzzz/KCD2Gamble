# Dice Game

这是一个双人回合制摇骰子游戏，支持命令行模式和 `tkinter` 图形界面。

## 运行方式

启动图形界面：

```bash
python dice_game.py --gui
```

启动命令行版本：

```bash
python dice_game.py
```

也可以通过参数指定目标分数和随机种子：

```bash
python dice_game.py --gui --target 3000 --seed 7
python dice_game.py --target 3000 --seed 7
```

## 图形界面说明

- 顶部可以设置目标分数和随机种子，并随时开始新游戏
- 左侧显示双方总分、当前回合暂存分数和待掷骰子数量
- 中间骰子区支持点击选择得分骰子
- 下方按钮分别用于掷骰、拿走并计分、继续掷剩余骰子、把本回合分数入账
- 右侧包含对局日志和完整计分规则

## 计分规则

- 一个 `1`：`100` 分
- 一个 `5`：`50` 分
- `1 2 3 4 5`：`500` 分
- `2 3 4 5 6`：`750` 分
- `1 2 3 4 5 6`：`1500` 分
- 三个 `1`：`1000` 分
- 三个 `2 / 3 / 4 / 5 / 6`：`200 / 300 / 400 / 500 / 600` 分
- 从三个相同点数开始，每多一个相同骰子，当前组合分数翻倍
